Problem Set 2 — Answer Key

1) Bias–Variance Tradeoff – Practical

Model A has higher variance (very low train error, large generalization gap: 5% → 20%).

Model B has higher bias (higher train error and only slightly higher test error: 15% → 17%).

2) Elastic Net – Parameter Effects

With mixing parameter α → 1, the model behaves more like Lasso (L1 dominates). Expect more coefficients driven exactly to zero (stronger feature selection) while a residual L2 component stabilizes correlated predictors.

3) Search Algorithm Identification

Sequential (linear) search; worst-case time complexity O(n).

4) Recursion – Coding Problem

Example solution in Python:

def sum_digits(n: int) -> int:
    if n < 10:
        return n
    return n % 10 + sum_digits(n // 10)

Works for any positive integer n. Base case: single digit; recursive step peels off last digit and recurses on the prefix.

5) Combinations and Permutations

(a) Committees (order irrelevant): C(8,3) = 8! / (3! 5!) = 56.

(b) Ordered selections: P(8,3) = 8 × 7 × 6 = 336 = 3! · C(8,3).
