Problem Set 2 (5 questions—includes one coding problem)

Topics covered: Bias–Variance tradeoff, Elastic Net, Sequential vs. Binary Search, Recursion (coding), Combinations & Permutations.

1) Bias–Variance Tradeoff – Practical

You test two models:

  • Model A: Train error = 5%, Test error = 20%
  • Model B: Train error = 15%, Test error = 17%

Which model appears to have higher variance? Which has higher bias? Briefly explain.

Expected answer type: Short conceptual explanation.

2) Elastic Net – Parameter Effects

Elastic Net uses both L1 and L2 penalties. If you set the mixing parameter α close to 1, which penalty dominates? What effect would you expect on feature selection?

Expected answer type: Short conceptual explanation.

3) Search Algorithm Identification
Start at the first element in the list, check each element one by one until the target is found or the list ends.

Name the search algorithm and give its worst-case time complexity in Big O notation.

Expected answer type: One-line identification + complexity.

4) Recursion – Coding Problem

Write a recursive Python function sum_digits(n) that takes a positive integer n and returns the sum of its digits.

# Example
sum_digits(1234)  # returns 10

Expected answer type: Python function definition written by you.

5) Combinations and Permutations

A committee of 3 people will be formed from a group of 8 candidates.

  1. (a) How many different committees can be formed?
  2. (b) How many different ways could the same 3 people be chosen if order matters?

Expected answer type: Numeric answers with formulas (combinations and permutations).
