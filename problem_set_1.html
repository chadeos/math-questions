Problem Set 1 (6 questions—no coding)

Topics covered: Bias–Variance tradeoff, Regularization, Gradient Descent (no momentum/Adam), Sorting, Big-O, Conditional Probability.

1) Bias–Variance Tradeoff (Conceptual)

You train a model and find that its performance on the training set is excellent, but on the test set it’s much worse.

  1. (a) Which side of the bias–variance tradeoff is this most likely showing?
  2. (b) Suggest two specific changes you could make to shift the model toward the opposite side of the tradeoff.

Expected answer type: Short conceptual explanation.

2) Ridge vs. Lasso (Conceptual)

Explain how the penalty term differs between Ridge regression and Lasso regression. Then, give one situation where Lasso might be preferred over Ridge.

Expected answer type: Short conceptual explanation.

3) Gradient Descent – Step Size Choice

You run gradient descent and notice that:

  • With a large learning rate, the loss oscillates wildly.
  • With a very small learning rate, the loss decreases slowly.

Explain why both issues occur and suggest a practical strategy for choosing the step size.

Expected answer type: Short conceptual explanation (no implementation).

4) Sorting – Merge Sort Analysis

Suppose you have a list of n integers. You decide to sort it using merge sort.

  1. (a) What is the time complexity in Big O notation?
  2. (b) Briefly explain why merge sort achieves this time complexity.

Expected answer type: Big-O identification and brief reasoning.

5) Big O – Function Comparison

Order the following functions from slowest growing to fastest growing as n → ∞:

n, n log n, n^2, 2^n, log n.

Expected answer type: Ordered list with a brief justification.

6) Probability – Conditional Probability

A box contains 3 red balls and 2 blue balls. You randomly draw one ball, do not replace it, and then draw a second ball. What is the probability that both balls are red?

Expected answer type: Numeric probability (fraction or decimal) with a short calculation.
